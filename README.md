# Contacts Manager web app

## Requirements:
- python flask
- pytest 
- selenium 

## Run the app
- clone the repo 
- run `flask run` at the root level
- open `http://127.0.0.1:5000/` in a browser

## Run the tests
- go to `/tests`
- run `pytest`
